Behind the AI
This specific AI bases oneâ€™s risk on your personal Blood Glucose, Blood Pressure, BMI and Age. Based on the data, we found the largest correlations between diabetes and these parameters. We also believed that these parameters were the most accessible to people, therefore increasing the usability of this AI.
This AI was trained on a data set from a hospital in Frankfurt, Germany using a random forest model. A random forest model runs the data set against many decision trees simultaneously to create an accurate model. A decision tree is a set of questions that become more specific and separate the data set into subcategories as you go down. As the random forest grows its number of trees, it becomes more trained on the data set and can start making predictions on new data sets.
Our Artificial Intelligence model has been optimized to achieve an accuracy rate of 93%. The percentage that our model outputs is the likelihood of it identifying the user as being diabetic. Any percentage above 50% means that our model would predict the user as being diabetic, with a higher percentage reflecting a greater confidence in the prediction.
